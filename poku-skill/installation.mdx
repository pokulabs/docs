---
title: "Poku Skill"
description: "Install and configure the Poku skill for any agent that supports the Agent Skills format, including Cursor, Claude Code, OpenCode, OpenAI Codex, and Pi."
---

## Prerequisite
Sign in to the [Poku dashboard](https://app.pokulabs.com) and generate an API key.

---

## Method 1: NPX Skills CLI

```bash
npx skills add pokulabs/skills
```

## Method 2: Cursor Marketplace

In Cursor, go to **Settings → Rules → Add Rule → Remote Rule (GitHub)** and enter:

```
pokulabs/skills
```

## Method 3: Manual Installation

Clone or download the repository at [pokulabs/skills](https://github.com/pokulabs/skills) and copy the `poku` folder to your agent's skills directory:

| Agent | Directory |
|-------|-----------|
| Claude Code | `~/.claude/skills/` |
| Cursor | `~/.cursor/skills/` |
| OpenCode | `~/.config/opencode/skills/` |
| OpenAI Codex | `~/.codex/skills/` |
| Pi | `~/.pi/agent/skills/` |

## Method 4: Clawhub (OpenClaw)

```bash
npx clawhub@latest install poku
```

For full setup details, see the [OpenClaw](/poku-skill/openclaw) page.

## Configuration

Set the following environment variables in your agent's configuration:

- **`POKU_API_KEY`** *(required)* — your Poku API key from [app.pokulabs.com](https://app.pokulabs.com)
- **`POKU_TRANSFER_NUMBER`** *(optional)* — phone number the agent will use to transfer calls to you, in E.164 format (e.g. `+12223334444`)

<Note>
  Installing the skill enables outbound calls and texts. To receive inbound calls and texts, a webhook setup is required. See the [Webhook](/poku-skill/webhook) page to get started.
</Note>
